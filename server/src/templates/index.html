CMPORT

Instructions

This is a tool that applies Modern Portfolio Theory to cryptocurrencies. To use it, input the cryptocurrencies you'd like to invest in as a coma-separated list of tickers and input the amount of previous days (including today) you'd like to gather the financial data from. Afterwards, press the submit button to run a statistical analysis on the data and get the results back.
